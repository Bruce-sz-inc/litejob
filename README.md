### LiteJob

A Fast & Safely Golang Task Dispatch.

### Features

* more storage engine support
    1. Redis(storage job list with redis list)
    2. Sqlite(storage job list with a table)
    3. Memory(storage job list with a memory map)
    4. Mysql
    ... others

* plugin easy
